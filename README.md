# 🌌 Zen-Drift

A minimalist, open-source web arcade game optimized for speed, mental relaxation, and focus. Built with absolute core technologies, it features a real-time global leaderboard and cross-device scaling.

## 🌐 Live Demo
Anyone can play the game instantly without installing anything. Click the official public web link below to play directly in your browser:

**👉 [Play Zen-Drift Live on Railway](https://zen-drift-production.up.railway.app/)**

## 🎯 Key Features
* **Flow State Design:** Zero aggressive graphics or stressful sounds. Engineered purely for anxiety relief and mindfulness.
* **Ultra-Fast & Lightweight:** Built using Vanilla JavaScript and HTML5 Canvas with no heavy frameworks, allowing the site to open instantly.
* **Smart Device Tracking:** Prevents duplicate entries on the leaderboard by tracking a unique device ID saved in the browser's local storage. A single player on the same device will only have their highest score updated, instead of creating multiple rows.
* **Real-time Leaderboard:** Automatically syncs scores across all users and devices globally in real-time.
* **Mobile-First Responsiveness:** Tailored explicitly for flawless touch-and-drag controls on modern mobile devices (including Android 10+) and standard mouse inputs on desktops.

## 🛠️ Technology Stack
* **Front-end:** Vanilla JavaScript (Canvas API), HTML5, CSS3
* **Back-end:** Node.js, Express.js
* **Database:** Neon Serverless PostgreSQL
* **Hosting Platform:** Railway
